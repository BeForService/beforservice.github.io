📖 How to Play

① Setup
Choose a Focus KPI — this KPI scores double points this session. Set team count (1-4) and names. Choose how many rounds.
② Each Round
A scenario card presents a real supply chain situation. Each team reads the same scenario and selects one of 3-4 strategic decisions simultaneously. Then decisions are revealed and KPIs update.
③ Scoring
Each decision impacts specific KPIs by +/- points. The Focus KPI impact counts double. Teams accumulate a cumulative score across all KPI movements. Highest total score wins.
④ Adding Content
Use the Scenarios tab to add custom scenarios. Each scenario needs a title, description, and 2-4 decisions with KPI impacts. Format: KPI_CODE:±delta e.g. RL.1.1:+8,CO.1.1:-5
⑤ Multiplayer
In a classroom: 1 screen, teams write their choice on paper, facilitator selects each team's choice in sequence. Or each team uses their own device — facilitator advances each turn. The game is designed so each team sees the same scenario and makes a blind choice before seeing others' results.
